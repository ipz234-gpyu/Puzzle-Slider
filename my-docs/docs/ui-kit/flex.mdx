# Layout Flexbox Container Utility

Flex is the shared layout primitive for composing rows, columns, spacing, and wrap behavior without reaching for ad hoc CSS in every feature.

## Import

```tsx
import { Flex } from '../../../src/shared/ui/flex/ui/Flex';
```

:::info
The `gap` prop accepts either a number in pixels or a string value. That makes it convenient for both design-token spacing and one-off layout tweaks.
:::

## Layout Parameters

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `direction` | `row \| column` | `row` | Controls the main axis direction. |
| `align` | `start \| center \| end \| stretch` | `stretch` | Controls cross-axis alignment. |
| `justify` | `start \| center \| end \| between \| around \| evenly` | `start` | Controls main-axis distribution. |
| `gap` | `number \| string` | unset | Adds spacing between children. |
| `wrap` | `nowrap \| wrap \| wrap-reverse` | `nowrap` | Controls whether items can wrap onto additional lines. |

## Notes

- Use `Flex` when you need a predictable inline layout that still accepts semantic HTML via the `as` prop.
- Prefer design-token spacing values or small pixel values for consistent component rhythm.
